# Time Series Analysis using ARIMA

## Objective
Perform Time Series Analysis on the AirPassengers Dataset using Statsmodels.

## Tools Used
- Python
- Pandas
- Matplotlib
- Statsmodels

## Dataset
AirPassengers Dataset

## Methodology
1. Load dataset
2. Visualize time series
3. Decompose trend and seasonality
4. Perform Augmented Dickey-Fuller (ADF) Test
5. Build ARIMA Model
6. Forecast next 12 months
7. Visualize forecast

## Deliverables
- Decomposition Plot
- Forecast Plot
- Forecast CSV

## Results
The dataset was analyzed for trend and seasonality. The ARIMA model was trained and used to forecast passenger counts for the next 12 months.

## Conclusion
Time Series Analysis helps identify trends, seasonality, and future patterns in temporal datasets, making it useful for forecasting applications.
